## üí¨ Communication Methods

### DM (Direct Message)
- Chat without mentions
- Fast response (2-3 seconds)
- Best for personal tasks

### Channel
- Requires `@botname` mention
- Visible to all team members
- Shared work context

### Thread
- Mention once when starting thread
- Continue without mentions after
- Organized topic-based conversations

---

## üë• Multi-User Support

### Per-User Memory
KIRA recognizes and remembers each user individually.

### Context Sharing
Everyone shares conversation context from channel discussions.

**Advantages:**
- Eliminates information gaps between team members
- No repeated questions needed
- Transparent decision-making

## üß† Memory Building Tips

Building memory systematically when first using KIRA makes it much more effective.

### Information to Remember

**Team Member Info:**
```
User: @KIRA Remember our team members
      @John Doe @Jane Smith @Bob Lee

KIRA: [Auto-collect via Slack MCP]
      Saved info for 3 team members to memory!
      (email, Slack ID, title, etc.)
```

**Organization structure, project info, Confluence docs** - Pre-loading enables:
- Auto-recognize email addresses without tagging
- Auto-reference project context
- Use information instantly without searching documents
