# 선제적 제안

KIRA가 메모리를 분석해서 필요한 작업을 먼저 제안합니다.

::: tip Beta 기능
선제적 제안은 현재 베타 버전입니다. 신중하게 활용하시기 바랍니다.
:::

## 🎯 작동 원리

### 자동 분석
설정된 주기(기본 15분)마다 로컬 메모리를 분석하고 패턴을 찾습니다.

---

## 🔍 7가지 개입 패턴

KIRA는 다음 패턴을 감지하고 제안합니다:

### 1. 📊 조사 제안
대화 중 정보가 부족할 때 자동으로 조사를 제안합니다.

### 2. ⏰ 스케줄링 제안
일정이나 마감일 논의 시 캘린더 등록을 제안합니다.

### 3. 📝 문서화 제안
중요한 결정이나 논의 후 문서화를 제안합니다.

### 4. ✍️ 초안 작성 제안
문서나 코드 작성이 필요할 때 초안을 제안합니다.

### 5. 🔗 연결 조율 제안
관련 있는 사람들을 연결하거나 정보를 공유합니다.

### 6. 🔮 예측적 제안
패턴 기반으로 다음 필요한 작업을 예측합니다.

### 7. 🤖 루틴 자동화 제안
반복적인 작업의 자동화를 제안합니다.

---

## ✅ 승인 프로세스

### 제안 승인
사용자가 "예/좋아/승인"이라고 응답하면 작업을 시작합니다.

### 제안 거부
"아니오/괜찮아/필요 없어"라고 응답하면 제안을 취소합니다.

---

## ⚙️ 설정

### 활성화 방법
1. KIRA 앱 > 환경변수 설정
2. **선제적 제안 기능** 섹션
3. **Proactive Enabled** 토글 켜기
4. 설정 저장 및 서버 재시작
