# 업무 시작하기

KIRA와 함께 업무를 시작하는 방법을 안내합니다.

## 🎯 모드 선택

KIRA는 **봇 모드**와 **가상직원 모드** 두 가지로 사용할 수 있습니다.

| 구분 | 봇 모드 | 가상직원 모드 |
|------|---------|---------------|
| **설치 위치** | 내 컴퓨터 | 전용 컴퓨터/VM |
| **Slack 계정** | 내 계정/봇 앱 | 가상직원 전용 계정 |
| **운영 시간** | 컴퓨터 켜져 있을 때 | 24/7 항시 가동 |
| **적합한 용도** | 개인 AI 비서 | 팀 공용 AI 직원 |

> 💡 **KRAFTON 사용 사례**: KRAFTON은 사내 계정과 컴퓨터를 지급하여 KIRA를 가상직원으로 운영하고 있습니다.

### 🤖 봇 모드
**내 컴퓨터**에 설치하고, **내가 직접** 관리하는 개인 AI 비서입니다.
- 내 컴퓨터가 꺼지면 KIRA도 중지됨
- 간단하게 시작하고 싶을 때 추천

### 👤 가상직원 모드
**전용 컴퓨터**에 설치하고, **24/7 독립적으로** 운영되는 팀 공용 AI 직원입니다.
- 가상직원 전용 Slack 계정 필요
- 실제 팀원처럼 운영하고 싶을 때 추천

::: tip 모드 설정 방법
모드별 상세 설정 방법은 [시작하기](/ko/getting-started#🎯-모드-선택) 페이지를 참고하세요.
:::

## 📋 기본 기능

### [대화하기](/ko/features/chat)
Slack에서 자연스럽게 대화하세요.
- DM, 채널, 스레드 어디서든
- 간단한 질문부터 복잡한 작업까지
- 자연어로 요청하면 됩니다

### [작업 수행](/ko/features/tasks)
KIRA가 처리할 수 있는 작업들입니다.
- 정보 검색 및 요약
- 문서 작성 및 번역
- 이메일/GitLab/Jira 관리

### [메모리 시스템](/ko/features/memory)
중요한 정보를 자동으로 기억합니다.
- 대화 내용 자동 저장
- 프로젝트 정보 관리
- 필요할 때 자동 참조

## 🤖 능동 수신 채널

::: tip Beta 기능
능동 수신 채널은 현재 베타 버전입니다. [설정 가이드](/ko/setup/)에서 활성화할 수 있습니다.
:::

### [이메일 모니터링](/ko/features/email-monitoring)
Outlook 이메일을 자동으로 확인하고 업무를 수행합니다.
- 설정된 주기(기본 5분)마다 이메일 확인
- 할당된 업무 자동 추출
- KIRA가 직접 업무 수행

### [Confluence 추적](/ko/features/confluence)
문서 업데이트를 자동으로 추적합니다.
- 설정된 주기(기본 60분)마다 업데이트 확인
- 중요 페이지만 요약
- 메모리에 자동 저장

### [Jira 추적](/ko/features/jira)
할당된 이슈를 자동으로 확인하고 업무를 수행합니다.
- 설정된 주기(기본 30분)마다 이슈 확인
- 할당된 업무 자동 추출
- KIRA가 직접 업무 수행

## 💡 사용 팁

### DM vs 채널
- **DM**: 개인 작업, 민감한 정보
- **채널**: 팀 공유, 투명한 의사결정

### 효과적인 요청
- 구체적으로 요청하세요
- 맥락을 충분히 제공하세요
- 긴 작업은 단계별로 나눠서

### 메모리 활용
- 중요한 정보는 명시적으로 기억하라고 요청
- 프로젝트 전용 채널 사용 권장
