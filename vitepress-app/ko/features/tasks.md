# 작업 수행

KIRA는 다양한 도구를 조합하여 복잡한 업무를 자동화합니다.

## 🎁 내장 기능 (설정 불필요)

KIRA는 별도 설정 없이 바로 사용할 수 있는 기능들을 내장하고 있습니다.

### 📄 문서 작성

| 기능 | 설명 | 예시 요청 |
|------|------|----------|
| **PDF** | PDF 문서 생성, 편집, 텍스트/표 추출 | "회의록을 PDF로 만들어줘" |
| **Excel (xlsx)** | 스프레드시트 생성, 수식, 데이터 분석 | "매출 데이터를 엑셀로 정리해줘" |
| **PowerPoint (pptx)** | 프레젠테이션 생성, 슬라이드 편집 | "프로젝트 소개 PPT 만들어줘" |
| **Word (docx)** | 문서 작성, 편집, 서식 지정 | "보고서를 워드 파일로 작성해줘" |

::: tip 파일 저장 위치
생성된 문서는 `~/Documents/KIRA/files/` 폴더에 저장되며, Slack으로 바로 전송할 수 있습니다.
:::

### 🔍 정보 검색 서비스

| 서비스 | 설명 | 예시 요청 |
|--------|------|----------|
| **Context7** | 최신 라이브러리/프레임워크 문서 검색 | "React 19 새 기능 알려줘" |
| **arXiv** | arXiv 논문 URL 분석 및 요약 | "이 논문 분석해줘: arxiv.org/abs/..." |
| **YouTube Info** | 유튜브 영상 정보 조회 | "이 유튜브 영상 요약해줘" |
| **Steam Review** | Steam 게임 리뷰 분석 | "이 게임 리뷰 분석해줘" |
| **Airbnb** | Airbnb 숙소 검색 | "서울 강남 근처 숙소 찾아줘" |
| **시간** | 현재 시간, 타임존 변환 | "뉴욕 지금 몇 시야?" |

::: info 추가 설정 없이 사용 가능
위 기능들은 KIRA 설치 시 자동으로 포함되어 있어 별도 API 키나 설정이 필요 없습니다.
:::

---

## 🛠️ MCP 도구 (설정 필요)

추가 설정을 통해 더 많은 도구를 사용할 수 있습니다. [설정 가이드](/ko/setup/)를 참고하세요.

| 도구 | 주요 기능 |
|------|----------|
| **Perplexity** | 실시간 웹 검색, 최신 정보 조회 |
| **MS365** | 이메일 읽기/전송, 캘린더, OneDrive, SharePoint |
| **Confluence** | 문서 검색/생성/수정 |
| **Jira** | 이슈 조회/생성/상태 변경 |
| **GitHub/GitLab** | 저장소, 이슈, PR/MR 관리 |
| **DeepL** | 문서 번역 |
| **X (Twitter)** | 트윗 작성/게시 |
| **Tableau** | 대시보드 조회, 데이터 시각화 |
| **Clova Speech** | 음성→텍스트 변환, 회의록 작성 |

## 💡 활용 사례

### 📊 주간 업무 자동화

**매주 금요일 주간 보고서 자동 생성**
```
사용자: 매주 금요일 오후 5시에 주간 보고서 작성해서 팀 채널에 공유해줘

KIRA: [금요일 17:00 자동 실행]
      1. Jira에서 이번 주 완료/진행 중 이슈 조회
      2. GitLab에서 이번 주 머지된 MR 목록 조회
      3. 메모리에서 주요 논의사항 검색
      4. 종합하여 주간 보고서 작성
      5. Confluence에 페이지 생성
      6. Slack 팀 채널에 요약 공유
```

**매일 아침 업무 브리핑**
```
사용자: 매일 오전 9시에 오늘 할 일 정리해서 알려줘

KIRA: [매일 09:00 자동 실행]
      1. Outlook에서 오늘 일정 조회
      2. Jira에서 나에게 할당된 이슈 조회
      3. 이메일에서 답장 필요한 건 확인
      4. 종합하여 오늘의 할 일 목록 생성
      5. DM으로 브리핑 전송
```

---

### 📧 이메일 기반 업무 처리

**이메일 요청 → Jira 티켓 생성 → 담당자 알림**
```
사용자: 고객사에서 온 버그 리포트 이메일 처리해줘

KIRA: 1. Outlook에서 버그 리포트 이메일 조회
      2. 내용 분석하여 Jira 이슈 생성
         - 제목: [고객사] 로그인 오류 발생
         - 우선순위: P1
         - 담당자: 개발팀
      3. Slack에서 담당자에게 알림
      4. 고객사에 접수 확인 이메일 발송
```

**다국어 이메일 자동 번역 및 답장**
```
사용자: 영어로 온 이메일 번역하고 한국어로 답장 초안 써줘

KIRA: 1. Outlook에서 영어 이메일 조회
      2. DeepL로 한국어 번역
      3. 답장 초안 작성 (한국어)
      4. DeepL로 영어 번역
      5. 이메일 초안으로 저장
```

### 📄 문서 작업 자동화

**회의록 → Confluence → Jira 액션 아이템**
```
[웹 인터페이스에서 회의 녹음 업로드]

KIRA: 1. Clova Speech로 음성 텍스트 변환
      2. 회의록 형식으로 정리
         - 참석자, 논의사항, 결정사항
      3. Confluence에 회의록 페이지 생성
      4. 액션 아이템 추출
      5. 각 담당자에게 Jira 이슈 생성
      6. Slack으로 회의록 링크 공유
```

**Confluence 문서 번역 및 공유**
```
사용자: API 문서를 영어로 번역해서 글로벌 팀에 공유해줘

KIRA: 1. Confluence에서 API 문서 조회
      2. DeepL로 영어 번역
      3. Confluence에 영문 페이지 생성
      4. 글로벌 팀 채널에 링크 공유
```

### 🔍 정보 수집 및 리서치

**경쟁사 분석 리포트**
```
사용자: 경쟁사 A의 최신 뉴스 조사해서 정리해줘

KIRA: 1. Perplexity로 경쟁사 A 최신 뉴스 검색
      2. 주요 내용 요약
      3. Confluence에 경쟁사 분석 페이지 업데이트
      4. 관련 팀에 Slack 알림
```

**기술 조사 후 문서화**
```
사용자: Next.js 15 새로운 기능 조사해서 팀 위키에 정리해줘

KIRA: 1. Perplexity로 Next.js 15 신기능 검색
      2. 공식 문서 내용 정리
      3. 우리 프로젝트에 적용 가능한 부분 분석
      4. Confluence에 기술 리서치 페이지 생성
```

### 📈 데이터 분석 및 보고

**Tableau 대시보드 분석 및 공유**
```
사용자: 이번 달 매출 대시보드 캡처해서 경영진에게 보고해줘

KIRA: 1. Tableau에서 매출 대시보드 조회
      2. 주요 지표 분석 (전월 대비, YoY)
      3. 대시보드 이미지 생성
      4. 분석 내용과 함께 이메일 작성
      5. 경영진에게 Outlook으로 발송
```

### 🔄 개발 워크플로우

**PR 리뷰 요청 자동화**
```
사용자: 내 PR 리뷰 요청해줘

KIRA: 1. GitLab에서 내 오픈된 MR 조회
      2. 변경 내용 요약
      3. 리뷰어 자동 지정 (메모리에서 팀 정보 참조)
      4. Slack으로 리뷰 요청 메시지 전송
```

**배포 후 알림 및 문서 업데이트**
```
사용자: v2.0 배포 완료 후 처리해줘

KIRA: 1. GitLab에서 v2.0 태그 및 변경사항 조회
      2. 릴리즈 노트 작성
      3. Confluence에 릴리즈 문서 업데이트
      4. Jira에서 관련 이슈들 Done 처리
      5. 팀 채널에 배포 완료 공지
```

### 📱 소셜 미디어 관리

**트윗 시리즈 예약**
```
사용자: 신제품 출시 관련 트윗 3개 만들어서 내일 오전에 1시간 간격으로 게시해줘

KIRA: 1. 신제품 정보 메모리에서 조회
      2. 트윗 3개 초안 작성
      3. 스케줄 등록:
         - 09:00 트윗 #1
         - 10:00 트윗 #2
         - 11:00 트윗 #3
      4. 게시 완료 후 Slack 알림
```

매월 1일 09:00 - 월간 KPI 리포트 (Tableau + Jira 통합)

매주 수요일 14:00 - 프로젝트 A 진행 상황 점검

2. Slack에서 KIRA에게 원하는 작업 요청
3. 반복 작업은 스케줄 등록으로 자동화

:::
