# 대화하기

KIRA와 Slack에서 자연스럽게 대화할 수 있습니다.

::: tip 모드 선택
KIRA는 **봇 모드**와 **가상직원 모드** 두 가지로 사용할 수 있습니다.
모드별 설정 방법은 [시작하기](/ko/getting-started#🎯-모드-선택) 페이지를 참고하세요.
:::

---

## 💬 대화 방식

### DM (Direct Message)
- 멘션 없이 바로 대화 가능
- 빠른 응답 (2-3초)
- 개인 작업에 최적

### 채널 (Channel)
- `@봇이름` 멘션 필요
- 팀원 모두가 볼 수 있음
- 작업 맥락 공유

### 스레드 (Thread)
- 스레드 시작 시 한 번만 멘션
- 이후 멘션 없이도 대화 가능
- 주제별 체계적인 대화

---

## 👥 멀티 유저 지원

### 사용자별 맞춤 메모리
KIRA는 각 사용자를 개별적으로 인식하고 기억합니다.

### 컨텍스트 공유
채널에서 다른 사용자와의 대화 맥락을 모두가 공유받습니다.

**장점:**
- 팀원 간 정보 격차 해소
- 반복 질문 불필요
- 투명한 의사결정

---

## 🧠 메모리 구축 팁

KIRA를 처음 사용할 때 메모리를 체계적으로 구축하면 훨씬 효과적입니다.

### 기억시켜야 할 정보

**팀원 정보:**
```
사용자: @KIRA 우리 팀원들 기억해줘
       @John Doe @Jane Smith @Bob Lee

KIRA: [Slack MCP로 자동 수집]
      3명의 팀원 정보를 메모리에 저장했습니다!
      (이메일, Slack ID, 직책 등)
```

**조직 구조, 프로젝트 정보, Confluence 문서 등**을 미리 기억시켜두면:
- 팀원 태그 없이도 자동으로 이메일 주소 인식
- 프로젝트 맥락을 자동으로 참조
- 문서를 찾지 않고도 즉시 정보 활용

::: tip 상세 가이드
자세한 메모리 구축 방법은 [메모리 시스템](/ko/features/memory#메모리-초기화-가이드) 페이지를 참고하세요.
:::
