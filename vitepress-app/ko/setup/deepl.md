# DeepL (문서 번역)

DeepL API를 연동하면 KIRA가 문서를 고품질로 번역할 수 있습니다.

## 📋 준비물

- DeepL API 계정
- API 키 (Free 또는 Pro)

---

## 🔑 1단계: API 키 발급

### 1. DeepL 계정 생성
1. [DeepL API 페이지](https://www.deepl.com/pro-api) 접속
2. **"Sign up for free"** 클릭
3. 이메일로 가입

### 2. API 플랜 선택
- **DeepL API Free**: 월 500,000자 무료
- **DeepL API Pro**: 유료 (사용량 기반)

### 3. API 키 복사
1. 가입 완료 후 대시보드 접속
2. **"API Keys"** 탭 클릭
3. **"Authentication Key"** 복사

::: warning API 키 종류
- **Free API**: `....:fx` 로 끝남
- **Pro API**: 일반 문자열

KIRA는 두 종류 모두 지원합니다.
:::

---

## ⚙️ 2단계: KIRA 설정

### 1. KIRA 앱 실행
환경변수 설정 탭을 엽니다.

### 2. DeepL 섹션 찾기
**MCP 설정** > **DeepL**

### 3. 설정 활성화
- 토글 스위치를 **켜기**로 변경

### 4. API 키 입력
- **DEEPL_API_KEY**: 앞서 복사한 API 키

### 5. 설정 저장
- **"설정 저장"** 버튼 클릭
- 서버 재시작

---

## ✅ 3단계: 테스트

Slack에서 KIRA에게 물어보세요:

```
"Hello, how are you?" 한국어로 번역해줘
```

```
"안녕하세요" 영어로 번역해줘
```

KIRA가 DeepL을 사용하여 번역합니다.

---

## 🎯 사용 예시

### 영어 → 한국어
```
사용자: "Machine learning is awesome" 한국어로 번역해줘
KIRA: [DeepL 번역] "머신러닝은 훌륭합니다"
```

### 한국어 → 영어
```
사용자: "이 문서를 영어로 번역해줘: 안녕하세요..."
KIRA: [DeepL 번역] "Hello..."
```

### 다국어 지원
```
사용자: "Bonjour" 일본어로 번역해줘
KIRA: [DeepL 번역] "こんにちは"
```

DeepL이 지원하는 언어:
- 영어, 한국어, 일본어, 중국어
- 독일어, 프랑스어, 스페인어, 이탈리아어
- 포르투갈어, 네덜란드어, 폴란드어, 러시아어
- 기타 30개 이상 언어

## 🔧 문제해결

### "Invalid API key"
- API 키가 올바르게 입력되었는지 확인
- Free API 키는 `:fx`로 끝나는지 확인
- DeepL 대시보드에서 API 키 상태 확인

### "Quota exceeded"
- 월 무료 한도(500,000자)를 초과했습니다
- DeepL 대시보드에서 사용량 확인
- Pro 플랜으로 업그레이드 고려

### 번역이 작동하지 않음
- DEEPL_ENABLED가 True로 설정되었는지 확인
- 서버를 재시작했는지 확인
- 로그에서 에러 메시지 확인
