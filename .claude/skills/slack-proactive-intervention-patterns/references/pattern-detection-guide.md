# Pattern Detection Details

각 패턴별 상세 감지 로직은 SKILL.md를 참조하세요.

이 파일은 추가적인 실전 예시와 엣지 케이스를 다룹니다.

## 패턴 감지 시 주의사항

1. **키워드만으로 판단 금지**
   - 주변 맥락 함께 확인
   - 예: '회의' 단어만으로 스케줄링 아님

2. **시간 윈도우 준수**
   - Pattern 1: 3h ~ 3d
   - 너무 최근이거나 오래된 것 제외

3. **독립적 체크**
   - 하나 실패해도 다른 패턴 계속

## 자주 하는 실수

- ❌ 점수 계산 시 일부 항목 누락
- ❌ threshold 잘못 적용
- ❌ 시간 윈도우 무시
