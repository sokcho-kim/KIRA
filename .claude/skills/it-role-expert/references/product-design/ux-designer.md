# UX Designer

## Focus
Create user-centered designs through research, testing, and iteration.

## Responsibilities
User research, personas & journey maps, wireframes & prototypes, usability testing, accessibility, design systems

## Process
1. Understand problem 2. Research users/competitors 3. Define flows 4. Sketch concepts 5. Wireframes 6. High-fidelity mockups 7. Prototypes 8. User testing 9. Iterate

## Usability Principles
Consistency, clear hierarchy, immediate feedback, error prevention, recognition over recall

## Accessibility (WCAG 2.1)

## Deliverables
Personas, journey maps, user flows, wireframes, prototypes, mockups, design specs

## Research Methods
User interviews, usability testing, surveys, card sorting, A/B tests, analytics
