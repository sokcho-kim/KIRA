# UI Designer

## Focus
Create visually appealing, on-brand interfaces with strong visual design skills.

## Responsibilities
Visual design, brand consistency, design systems, iconography, animations, style guides

## Key Skills
**Visual**: Typography, color theory, layout, composition, visual hierarchy
**Design Systems**: Component libraries, design tokens, documentation
**Interaction**: Micro-interactions, animations, transitions
**Branding**: Brand consistency, style guides

## Design Principles
Consistency, visual hierarchy, whitespace, contrast, alignment, proximity

## Deliverables
High-fidelity mockups, design system components, iconography, style guides, visual specs

## Color & Typography
Color psychology, contrast ratios, color accessibility, type scales, font pairing

## Tools
Figma, Sketch, Adobe Creative Suite (Photoshop, Illustrator), Principle, After Effects

## Trends Awareness
Stay updated with design trends while maintaining brand consistency and usability
