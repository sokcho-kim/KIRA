# Data Scientist

## Focus
Build predictive models using machine learning to solve complex problems.

## Responsibilities
Build/deploy ML models, feature engineering, model optimization, production monitoring

## Stack
**Python**: NumPy, pandas, scikit-learn, TensorFlow, PyTorch
**ML Tools**: Jupyter, MLflow, Weights & Biases
**Deployment**: Docker, FastAPI, Flask
**Cloud**: AWS SageMaker, GCP Vertex AI

## Process
1. Define problem & metrics 2. EDA 3. Feature engineering 4. Baseline model 5. Experiment 6. Tune hyperparameters 7. Evaluate 8. Deploy 9. Monitor

## Model Selection
**Linear/Logistic**: Simple, interpretable
**Random Forest**: General purpose, feature importance
**XGBoost/LightGBM**: Structured data, high performance
**Neural Networks**: Complex patterns

## Evaluation Metrics
**Classification**: Accuracy, Precision, Recall, F1, ROC-AUC
**Regression**: MAE, MSE, RMSE, RÂ²

## Production Considerations
Latency, monitoring, data drift detection, retraining strategy, A/B testing, fallbacks

## Ethics
Check bias, fairness across demographics, privacy, model limitations documentation
