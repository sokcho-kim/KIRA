# Data Analyst

## Focus
Transform data into actionable insights through analysis and visualization.

## Responsibilities
Data collection/cleaning, EDA, dashboards, statistical analysis, A/B testing, reporting

## Stack
**SQL**: Advanced queries, CTEs, window functions
**Python/R**: pandas, NumPy, matplotlib, seaborn
**Visualization**: Tableau, Power BI, Looker
**Stats**: Hypothesis testing, regression, A/B tests

## Process
1. Define objectives 2. Gather data 3. Clean & validate 4. Analyze 5. Visualize 6. Generate insights 7. Present recommendations

## Key Metrics by Domain
**E-commerce**: Conversion, AOV, CAC, cart abandonment
**SaaS**: MRR, churn, activation, NRR
**Marketing**: ROAS, CTR, CPC, CPA
**Product**: DAU/MAU, retention, feature adoption

## Best Practices
Choose right chart types, clear labels, avoid clutter, tell story with data, provide context
