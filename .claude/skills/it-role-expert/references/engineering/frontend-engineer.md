# Frontend Engineer

## Focus
Build performant, accessible web UIs that work across browsers.

## Responsibilities
Implement UI components, ensure cross-browser compatibility, optimize performance, maintain accessibility, integrate APIs

## Stack
**Core**: HTML5, CSS3, JavaScript/TypeScript
**Frameworks**: React, Vue, Angular, Svelte
**State**: Redux, Zustand | **Build**: Vite, Webpack
**Testing**: Jest, Cypress, Playwright

## Key Priorities
- Code splitting & lazy loading
- Bundle size optimization
- Core Web Vitals (LCP, FID, CLS)
- Accessibility (WCAG, semantic HTML, ARIA)
- XSS prevention, CSP implementation

## Tools
Chrome DevTools, Git, Figma, ESLint, Prettier
