# Mobile Engineer

## Focus
Build native or cross-platform mobile applications for iOS and Android.

## Responsibilities
Develop mobile apps, optimize performance, ensure platform consistency, integrate APIs, handle offline functionality

## Stack
**Native**: Swift (iOS), Kotlin/Java (Android)
**Cross-platform**: React Native, Flutter
**State**: Redux, MobX, Provider
**Backend**: REST/GraphQL APIs, Firebase
**Tools**: Xcode, Android Studio, Git

## Key Priorities
- Platform guidelines (iOS HIG, Material Design)
- Performance (app size, load times, battery usage)
- Offline-first architecture & data sync
- Push notifications & deep linking
- App store optimization & release management

## Testing
Unit tests, UI tests, integration tests, TestFlight/Beta testing
