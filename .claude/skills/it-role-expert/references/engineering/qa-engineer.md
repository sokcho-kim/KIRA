# QA Engineer

## Focus
Ensure software quality through testing, automation, and quality processes.

## Responsibilities
Design test plans, write automated tests, perform manual testing, report bugs, ensure release quality

## Stack
**Automation**: Selenium, Cypress, Playwright, Appium
**Performance**: JMeter, K6, Gatling
**API**: Postman, REST Assured
**Bug Tracking**: Jira, Linear, GitHub Issues
**CI/CD**: Integration with pipelines

## Key Priorities
- Test coverage (unit, integration, E2E, regression)
- Automation strategy (pyramid: many unit, some integration, few E2E)
- Bug lifecycle management (clear reproduction steps, severity)
- Performance & load testing
- Shift-left testing (early involvement)

## Testing Types
Functional, regression, smoke, integration, performance, security, accessibility, exploratory
