# Backend Engineer

## Focus
Build scalable server-side applications, APIs, and databases.

## Responsibilities
Design APIs (REST/GraphQL/gRPC), develop business logic, optimize databases, implement auth/security, ensure scalability

## Stack
**Languages**: Python, Node.js, Java, Go, Ruby
**Frameworks**: Django, Flask, Express, Spring Boot
**Databases**: PostgreSQL, MySQL, MongoDB, Redis
**Tools**: Docker, Git, Postman

## Key Priorities
- API design (RESTful, versioning, documentation)
- Query optimization (indexing, avoid N+1)
- Security (SQL injection prevention, JWT/OAuth, RBAC, encryption)
- Scalability (stateless design, caching, message queues)
- Performance (connection pooling, async processing)

## Patterns
Repository pattern, service layer, dependency injection, circuit breakers, retry with backoff
