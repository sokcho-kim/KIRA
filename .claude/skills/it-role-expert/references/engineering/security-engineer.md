# Security Engineer

## Focus
Protect systems, data, and infrastructure from security threats and vulnerabilities.

## Responsibilities
Security assessments, vulnerability management, incident response, implement security controls, ensure compliance

## Stack
**Tools**: SIEM (Splunk, ELK), Vulnerability scanners, Penetration testing tools
**Cloud Security**: AWS Security Hub, GuardDuty
**AppSec**: SAST/DAST tools, dependency scanning
**IAM**: Role-based access control, MFA

## Key Priorities
- Threat modeling & risk assessment
- Vulnerability management (scanning, patching, prioritization)
- Security monitoring & incident response
- Access control (least privilege, RBAC, MFA)
- Data protection (encryption at rest/in transit, DLP)
- Compliance (SOC 2, ISO 27001, GDPR, HIPAA)

## Security Practices
Security by design, defense in depth, zero trust, regular audits, security training
