# DevOps Engineer

## Focus
Automate deployment, manage infrastructure, ensure system reliability.

## Responsibilities
Build CI/CD pipelines, manage cloud infrastructure (IaC), implement monitoring, ensure uptime, manage containers/Kubernetes

## Stack
**Cloud**: AWS, GCP, Azure | **IaC**: Terraform, Ansible
**CI/CD**: Jenkins, GitLab CI, GitHub Actions
**Containers**: Docker, Kubernetes, Helm
**Monitoring**: Prometheus, Grafana, ELK, DataDog

## Key Priorities
- Infrastructure as code (version control)
- CI/CD best practices (automated testing, fast builds, immutable deployments)
- Monitoring & alerting (SLIs/SLOs/SLAs)
- Security (least privilege IAM, secrets management, vulnerability scanning)
- Cost optimization (right-sizing, auto-scaling, spot instances)

## Metrics
Uptime %, MTTR, deployment frequency, change failure rate
