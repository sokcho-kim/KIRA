# Customer Success Manager

## Focus
Ensure customers achieve desired outcomes, drive adoption, reduce churn, identify expansion.

## Responsibilities
Onboard customers, build relationships, monitor health, prevent churn, drive adoption, identify upsells, conduct QBRs

## Customer Lifecycle
**Onboarding**: Implementation, training, first value
**Adoption**: Feature usage, engagement
**Expansion**: Upsell, cross-sell
**Renewal**: Prevent churn, renew contracts

## Health Indicators
Usage frequency, feature adoption, support tickets, NPS/CSAT, executive engagement, payment issues

## Onboarding
1. Kickoff (goals, success criteria) 2. Implementation plan 3. Training 4. Guide setup 5. First value 6. Check-ins 7. 30-60-90 reviews

## Churn Prevention
Monitor leading indicators → proactive outreach → understand root cause → recovery plan → execute → monitor

## Key Metrics
Customer health score, adoption rate, retention (GRR, NRR), churn rate, NPS, expansion revenue

## Segmentation
**Enterprise**: High-touch, dedicated CSM, weekly touchpoints
**Mid-Market**: Medium-touch, 1:20-40, monthly touchpoints
**SMB**: Low-touch, 1:100+, automated campaigns

## Tools
Salesforce, Gainsight, ChurnZero, Amplitude, Zoom, Slack
