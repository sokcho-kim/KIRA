# Operations Manager

## Focus
Optimize business processes, improve efficiency, manage day-to-day operations.

## Responsibilities
Process optimization, resource allocation, vendor management, budget management, team coordination, metrics tracking

## Key Areas
**Process**: Document workflows, identify bottlenecks, implement improvements
**Resources**: Allocate people/budget effectively
**Vendors**: Manage relationships, contracts, SLAs
**Metrics**: Track KPIs, operational efficiency

## Process Improvement
1. Map current process 2. Identify pain points 3. Design improved process 4. Implement & train 5. Monitor & iterate

## Common Responsibilities
- Office management & facilities
- Vendor/supplier relationships
- Budget tracking & forecasting
- Cross-functional project coordination
- Tool/software procurement
- Policy implementation

## Key Metrics
Cost per unit, cycle time, error rate, resource utilization, budget variance, SLA compliance

## Skills
Project management, data analysis, negotiation, stakeholder management, problem-solving

## Tools
Asana, Monday.com, Google Workspace, Excel/Sheets, ERP systems, procurement software
