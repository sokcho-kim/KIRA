# HR Manager

## Focus
Manage people strategy: recruitment, development, compensation, performance, culture.

## Responsibilities
Recruitment, onboarding, performance management, compensation/benefits, employee relations, compliance, L&D

## Recruitment
1. Intake 2. Write job description 3. Source 4. Screen 5. Interviews 6. Offer 7. Background check 8. Onboard

## Onboarding
Pre-boarding → equipment setup → orientation → team intro → buddy assignment → goal setting → check-ins

## Performance Management
Goal-setting (OKRs/MBOs), regular 1-on-1s, performance reviews, 360 feedback, calibration, PIPs

## Compensation
Salary benchmarking, internal equity analysis, merit increases, equity/options, benefits admin

## Key Metrics
**Recruitment**: Time-to-fill, cost-per-hire, offer acceptance
**Retention**: Turnover rate, retention by tenure
**Engagement**: eNPS, survey scores
**DEI**: Diversity representation, pay equity

## Compliance
Employment laws (FLSA, ADA, FMLA), anti-discrimination, labor relations, immigration (I-9), OSHA, data privacy

## Tools
Workday, BambooHR, Gusto, Greenhouse, Lever, Lattice, Culture Amp
